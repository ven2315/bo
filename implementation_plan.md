# 20代女性向けおしゃれカフェLP 実装計画書

## 1. プロジェクト概要
**ターゲット**: 20代女性、カフェ巡りや「映え」スポットを好む層、リラックスした時間を求める層。
**コンセプト**: "Urban Escape" (都会の隠れ家)
**デザインキーワード**: 
- **Modern & Chic**: ベージュ、ラテカラーを基調とした落ち着きのある配色。
- **Airy & Light**: 余白をたっぷりと取り、窮屈さを感じさせないレイアウト。
- **Interactive**: スクロールやホバーに合わせて心地よく動くマイクロインタラクション。

## 2. 現状分析
既存のプロジェクト（`src`ディレクトリ内）には、基本的なHTML構造とスタイル、JavaScriptによるフェードインアニメーションが実装されています。画像アセットも`public/images`に揃っています。
**改善ポイント**:
- より「プレミアム感」を出すためのタイポグラフィの微調整。
- 単調になりがちなスクロール体験をより豊かにするパララックス効果や画像のズームインエフェクト。
- モバイルメニューのデザインと挙動の洗練。

## 3. 実装フェーズ

### Phase 1: ビジュアル＆UIデザインのブラッシュアップ (CSS)
**目的**: 第一印象で「おしゃれ」と思わせるクオリティに引き上げる。
- [ ] **タイポグラフィ**: 欧文フォント(`Cormorant Garamond`)と和文フォント(`Noto Sans JP`)のバランス調整。カーニング（文字間隔）を広めにとり、洗練された印象に。
- [ ] **カラーパレット**: 既存のベージュに加え、深みのあるブラウンやアクセントのゴールドを効果的に配置し、メリハリをつける。
- [ ] **ボタンデザイン**: 「Ghost Button」（背景なし枠線のみ）や、ホバー時に色が反転するアニメーションを追加。
- [ ] **画像装飾**: 画像に微妙なドロップシャドウや、角丸（border-radius）の調整を行い、柔らかさを演出。

### Phase 2: インタラクションとアニメーションの強化 (JS/CSS)
**目的**: ユーザーを飽きさせない、動的な体験を提供する。
- [ ] **ヒーローセクション**: ページ読み込み時のオープニングアニメーション（ロゴがふわっと浮き出るなど）。
- [ ] **パララックス効果**: スクロールに合わせて背景画像や要素が異なる速度で移動する視差効果の実装。
- [ ] **スクロール連動**: 
    - メニュー画像のスクロールに合わせたズームイン。
    - テキストのラインごとの表示アニメーション。
- [ ] **モバイルメニュー**: ハンバーガーメニュークリック時の全画面オーバーレイメニューの実装（ぼかし背景など）。

### Phase 3: レスポンシブ＆最終調整
**目的**: どのデバイスでも崩れない美しさを保証する。
- [ ] **スマホ表示最適化**: 余白（パディング）の調整、フォントサイズの最適化。
- [ ] **SEO対策**: メタタグ、alt属性の最終確認。
- [ ] **パフォーマンス**: 画像の遅延読み込み（Lazy Loading）など。

## 4. 技術スタック
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **Tooling**: Vite (開発・ビルドツール)
- **Deployment**: GitHub Pages

## 5. 次のステップ
まずは **Phase 1: ビジュアル＆UIデザインのブラッシュアップ** から着手します。特にタイポグラフィと余白の調整を行い、ベースのデザイン品質を向上させます。
